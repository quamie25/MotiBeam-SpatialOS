IF SOMETHING BROKE
===============================================================================

1. Stop testing immediately
2. Note what broke (be specific)
3. Restore backup: ./safety.sh restore
4. Choose most recent backup from list
5. Test that restored version works
6. Report what broke to person who made changes

COMMON ISSUES:

Emoji showing as brackets:
  - Font loading broke
  - Restore backup immediately
  - Check load_emoji_font() function

Navigation not working:
  - Event handling broke
  - Restore backup
  - Check handle_key() and input methods

Realms won't open:
  - State management broke
  - Restore backup
  - Check enter_realm() and navigation_stack

App crashes on launch:
  - Syntax error or import error
  - Run: ./safety.sh test
  - Restore backup if syntax fails
